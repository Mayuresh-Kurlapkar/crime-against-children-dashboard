# Crime Against Children Analysis Dashboard

## Project Overview

Developed an interactive Power BI dashboard to analyze crime against children data from 2001–2012.

## Features

* Data Cleaning & Transformation
* Unpivoting using Power Query
* KPI Cards
* Heatmap Matrix
* Treemap Analysis
* Drill-through Pages
* Forecasting
* Interactive Slicers & Tooltips

## Tools Used

* Power BI
* Power Query
* DAX
* Excel

## Insights

* Identified states with highest crime rates
* Analyzed yearly crime trends
* Compared crime categories across states
